

******************** Proyecto TiendaAngular2  ***************************

1.- Importar la carpeta del proyecto en Visual Studio Code

1.- Instalar dependencias del Proyecto
	npm install
	
2.- credenciales de acceso
	usuario: romel@gmail.com
	clave: hola

3.- La base de datos del proyecto ya se encuentra en linea en Google Firebase, se adjunta
 una copia en un archivo con formato json	

4.- Arrancar el proyecto de la siguiente manera
	a.- desde la consola de comandos ejecutar ng serve
	b.- el link de la aplicacion es http://localhost:4200
